# Checklist Blocs Mac mini

Petit outil web statique pour suivre les blocs d'intervention depuis un téléphone.

## Fonctionnalités

- Checklist par bloc : configuration Mac mini, GitHub/code, Adyen
- Progression globale et progression par bloc
- Sauvegarde automatique dans le navigateur avec `localStorage`
- Notes rapides
- Interface responsive mobile

## Utilisation

Ouvrir `index.html` dans un navigateur, ou publier le dossier sur GitHub Pages / Netlify / Vercel.
